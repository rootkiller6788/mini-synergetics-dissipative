# Course Alignment — Order Parameters & Emergence

## Nine-School Curriculum Mapping

### MIT

- **8.334 Statistical Mechanics II**: Landau-Ginzburg theory, phase transitions
- **18.357 Nonlinear Dynamics**: Bifurcation theory, normal forms
- Mapping: L3 (GL theory), L4 (bifurcation), L6 (Kuramoto)

### Stanford

- **CS 229**: PCA for order parameter identification

- Mapping: L3 (Fokker-Planck), L5 (PCA), L4 (bifurcation control)
